## Operations Practiced

### Column Creation and Conditional Expressions

* Added constant columns using `lit()`.
* Created conditional columns using `when()` and `otherwise()`.
* Classified employees based on salary.
* Classified employees based on age.
* Classified employees based on experience.
* Classified employees based on department.

### String Processing

* Extracted text using `substring()`.
* Modified strings using `regexp_replace()`.
* Filtered records using `LIKE`.
* Retrieved records using `isin()`.

### Range-Based Filtering

* Used `between()` to filter values within a specified range.
* Filtered employees based on salary.
* Filtered employees based on age.
* Filtered employees based on joining year.
* Filtered employees based on employee ID.

### Data Reshaping

* Created summary reports using `pivot()`.
* Converted pivoted data back into rows using `unpivot()`.
* Restructured datasets using `stack()`.

### Window Functions

* Ranked records using `rank()`.
* Generated continuous rankings using `dense_rank()`.
* Assigned sequential numbers using `row_number()`.
* Retrieved previous row values using `lag()`.
* Retrieved next row values using `lead()`.
