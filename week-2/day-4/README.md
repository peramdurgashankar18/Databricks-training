# Day 4: StrDataNum SQL Assignment

## Assignment Overview

On Day 4, I worked on the **StrDataNum SQL Assignment**, which combines string, date, and numeric functions.

The main goal of this assignment was to apply the SQL functions learned during the previous days to practical data manipulation and transformation tasks.

## Assignment Objectives

Through this assignment, I practiced:

* Manipulating and formatting string data.
* Working with dates and timestamps.
* Performing numerical calculations.
* Combining multiple SQL functions.
* Writing queries using nested functions.
* Cleaning and transforming data.

## Scenario

The assignment focuses on working with a company database containing customer information.

The tasks involved:

1. Formatting and cleaning customer data.
2. Performing numerical calculations.
3. Processing date values for reports.
4. Creating formatted output using multiple SQL functions.
5. Combining string, date, and numeric operations in SQL queries.

## SQL Functions Used

### String Functions

The following string functions were practiced:

* `UPPER()`
* `LOWER()`
* `CONCAT()`
* `SUBSTRING()`
* `REPLACE()`
* `TRIM()`
* `LENGTH()`

These functions were used to clean, format, and manipulate text data.

### Date Functions

The following date functions were used:

* `YEAR()`
* `MONTH()`
* `DAY()`
* `DATEDIFF()`
* `DATE_ADD()`
* `DATE_FORMAT()`

These functions helped in extracting date information and performing date-based calculations.

### Number Functions

The numeric functions practiced include:

* `ROUND()`
* `CEIL()`
* `FLOOR()`
* `MOD()`
* `POWER()`
* `ABS()`

These functions were used for mathematical calculations and numeric data transformation.

## Assignment Approach

The assignment started with simple queries using individual SQL functions.

As the tasks progressed, multiple functions were combined to create more advanced queries. Nested function calls were also used to perform multiple data transformations within a single SQL query.

## Key Concepts Learned

* Combining multiple SQL functions in a single query.
* Using nested functions for data transformation.
* Cleaning and formatting customer information.
* Performing date-based calculations.
* Applying mathematical functions to numerical data.
* Creating readable and formatted query results.
